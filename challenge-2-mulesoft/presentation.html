How It Works

  • Every stored proc call uses correct parameter names and SQL types from the MuleSoft XML
  • Query-parameter routing replicates DataWeave logic in TypeScript
  • SHOT module bridges SOAP/WSDL to SABRE (haemovigilance system)
  • Swagger/OpenAPI auto-generated at /api/docs
  • Auth guard, structured logging, correlation ID tracing
  • 77 TypeScript files, 0 compilation errors
  • Address module: 60 automated tests passing (unit + integration)
